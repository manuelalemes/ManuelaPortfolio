Projeto ABP 2SEM - Sistema de Visualização e Disseminação de Dados Limnológicos

Descrição: Aplicação web desenvolvida para visualizar e disponibilizar dados limnológicos coletados pelo INPE em parceria com UFRJ, UFJF e IIE. O sistema integra informações de campanhas e do SIMA, permitindo acesso aberto a dados ambientais utilizados em estudos sobre o Balanço de Carbono nos reservatórios de Furnas.
Tecnologias: React, TypeScript, Node.js, PostgreSQL, Leaflet, Docker.

Minha participação: Atuei no desenvolvimento do sistema integrando frontend e backend, contribuí na organização das rotas, modelos e exibição das informações limnológicas. Também participei das decisões do time e na melhoria do produto.
